## Env

Required env vars:

```
PORT=...

FORTUBE_API_TOKEN=...  // session token
FORTUBE_HPY=...        // harvests per year: default: 50

FRY_HPY=...            // harvests per year. default: 365
FRY_WBNB_APY..         // pre compound APY. default: 90
FRY_BUSD_APY..         // pre compound APY. default: 90
```
